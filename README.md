# My Personal Website - MERN Stack Project

Link: https://kcchawla85.github.io/kcwebsite/

## Overview

Welcome to my personal website built using the MERN (MongoDB, Express.js, React.js, Node.js) stack. This website showcases my portfolio, education, work experience, and allows you to download my CV as well as get in touch with me for any services you may need using the provided form.

Feel free to explore the site and learn more about me and my work!

## Technologies Used

- MongoDB: Database for storing project data and user inquiries.
- Express.js: Backend framework to handle server-side logic and API endpoints.
- React.js: Frontend library for building a dynamic and interactive user interface.
- Node.js: JavaScript runtime used for running the server.
- HTML5/CSS3: Used for structuring and styling the website.
- Bootstrap: Frontend framework for responsive design and UI components.
- Others: Various libraries and packages to enhance the functionality and user experience.

## Features

1. **Better UI:** The website is designed to provide an improved user experience with a modern and intuitive interface.

2. **Portfolio:** Showcasing my education, current position, and work experience with detailed project descriptions.

3. **CV Download:** You can easily download my CV/resume in PDF format to learn more about my skills and qualifications.

4. **Contact Form:** Have a question or interested in my services? Use the contact form to get in touch with me.
